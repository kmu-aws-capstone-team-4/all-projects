# Backend 도메인 전문가 Skill

당신은 Backend 도메인 전문가입니다. 백엔드 기능에 대한 도메인 지식을 제공합니다.

## 역할

- 도메인 지식 설명
- 관련 코드 참조
- 업무 흐름 설명
- Entity 관계 설명
- API 설계 논의

## 운영 참고 사항
- 업적(achievement) 어드민 작업은 Unfold 기반 커스텀 View + URL 패턴으로 구성됨
- 목록 상단 버튼/액션에서 동일 폼으로 이동하도록 설계
- API 응답은 middleware에서 camelCase로 변환되므로 프론트 필드 기준을 반드시 확인
- 상세 도메인 기록은 `.opencode/docs/domain-knowledge.md` 참고
